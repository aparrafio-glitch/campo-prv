Por que falla el GPS
Chrome y Edge bloquean el GPS cuando el archivo se abre como file://. Es una restriccion de seguridad del navegador que no se puede desactivar con permisos.

Solucion: servir el archivo via HTTP desde la misma red.
Pasos en Windows
1
Pon los archivos en la misma carpeta

diagnostico_campo_PRV_v2.html y servidor_campo.py

2
Shift + clic derecho en la carpeta

"Abrir ventana de PowerShell" o "Abrir terminal"

3
Ejecuta el servidor
python servidor_campo.py
toca para copiar

Aparecera la URL del movil: http://192.168.X.X:8080/...

4
Conecta el movil al mismo WiFi

Abre esa URL en Chrome del movil. GPS funcionara correctamente.

Pasos en macOS
1
Abre Terminal en la carpeta

Arrastra la carpeta al icono de Terminal

2
Ejecuta
python3 servidor_campo.py
toca para copiar
3
Sin servidor_campo.py
python3 -m http.server 8080
toca para copiar

Abre: http://IP:8080/diagnostico_campo_PRV_v2.html

Metodo 1 - Simple HTTP Server (recomendado)
1
Instala "Simple HTTP Server" de Phlox
2
Configura la carpeta

Abre la app, toca el icono de carpeta, navega a Descargas y seleccionala.

3
Inicia el servidor

Toca Start. Anota el puerto que aparece (ej. 8080).

4
Abre en Chrome
http://localhost:8080/diagnostico_campo_PRV_v2.html
toca para copiar

GPS funcionara correctamente desde localhost.

5
Si Chrome bloquea la ubicacion

Chrome Menu (3 puntos) > Configuracion > Configuracion del sitio > Ubicacion > Busca localhost > Permitir

Configuracion Motorola
!
Activar GPS alta precision

Ajustes > Ubicacion > Precision de ubicacion > activar "Usar GPS, Wi-Fi, Bluetooth y redes moviles"

!
Permisos de Chrome

Ajustes > Aplicaciones > Chrome > Permisos > Ubicacion > "Permitir todo el tiempo"

!
Bateria (importante)

Ajustes > Bateria > Optimizacion de bateria > Chrome > No optimizar

Metodo 2 - GPS Manual (sin instalar nada)
1
Abre Google Maps

Manten presionado en tu posicion actual hasta ver un pin rojo.

2
Copia las coordenadas

Toca el panel inferior donde aparecen los numeros (ej. -33.439, -70.627). Manten presionado > Copiar.

3
Pegaas en el formulario

Toca "Ingresar manual" en el bloque GPS del formulario y pega la latitud y longitud por separado.

Precision: Google Maps usa GPS + WiFi + redes moviles. En terreno abierto la precision es 3-10 metros, suficiente para identificar el poste.
